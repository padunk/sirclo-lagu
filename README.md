# LAGU

## Getting Started

Clone the project

```bash
git clone git@github.com:padunk/sirclo-lagu.git
cd sirclo-lagu
```

Install dependencies with yarn or npm

```bash
yarn
```

Run the dev server at `localhost:3000`

```bash
yarn dev
```

Run test with Testing Library and Jest

```bash
yarn test
```
